# Рекомендуемая структура файлов

Структура файлов должна быть простой и масштабируемой, с четким разделением ответственности.

```
/src
├── /components         # Переиспользуемые React-компоненты
│   ├── /Button
│   │   ├── Button.tsx
│   │   └── Button.module.css
│   ├── /Display
│   │   ├── Display.tsx
│   │   └── Display.module.css
│   └── /ButtonGrid
│       ├── ButtonGrid.tsx
│       └── ButtonGrid.module.css
│
├── /core               # Ядро калькулятора (бизнес-логика)
│   └── CalculatorCore.ts
│
├── /hooks              # Пользовательские хуки (если понадобятся)
│   └── useCalculator.ts  # Пример: хук для инкапсуляции useReducer
│
├── /services           # Вспомогательные сервисы (логгер и др.)
│   └── Logger.ts
│
├── /styles             # Глобальные стили
│   └── global.css
│
├── /types              # Глобальные типы TypeScript
│   └── index.ts
│
├── App.tsx             # Главный компонент приложения
├── index.tsx           # Точка входа
└── react-app-env.d.ts  # Декларации типов для CRA
```
